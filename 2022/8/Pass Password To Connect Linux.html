Pass Password To Connect Linux

Published Date: 2022-08-13 16:04:35Z

Environment

  • ubuntu 20.04

Steps

  1. Generated your key
    • ssh-keygen -t rsa
  2. Copy public key file *.pub to remote
    • scp {new public key} remote:~/.ssh
  3. Add to auth
    • login remote
    • cd ~/.ssh
    • cat {your public key} >> authorized_keys
  4. Try to reconnected to remote and found it done

Extra

Maybe it is not success after these steps when your client is windows. You should make a file config in your folder {profile}/.ssh/. Content(without {}):

Host {NameYouWant}
    User {UserName}
    HostName {HostNameOrIp}
    Port {22}
    IdentityFile {C:/Users/Administrator/.ssh/id_anest_github}

End
