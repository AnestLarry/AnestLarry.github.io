Wsl2 Mount Windows Disk Drive

Published Date: 2021-08-24 11:04:35Z

Background

  • Access Windows files in a disk drive without a drive letter in wsl2.
  • The disk drive has a drive path at least.

Action

  • run sudo mount -t drvfs <win_path> <wsl2_path> in your wsl2.
    • win_path: the Windows drive path you want.
    • wsl2_path: the wsl2 path you want.
    • e.g: sudo mount drvfs C:\Data /mnt/Data
      • C:\Data need created at before.(mkdir C:\Data)
      • /mnt/Data need created at before.(mkdir /mnt/Data)
      • then, the files can access in wsl2 /mnt/Data.

Other

  • docker with wsl2 also could access those files by the way.
