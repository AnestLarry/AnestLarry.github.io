Unity : Make a simple game

读完这篇教程而记 https://www.cnblogs.com/OctoptusLian/p/8418534.html

前言 unity版本2018.1.3.f1 安装unity的时候出现了个坑,以前安装过unity,license 认证死活过不去,然后重装解决了这个问题 upgrade license上依然出错,解决方法: 1 log out后再login 2 删掉C:/ProgramData中的Unity文件夹(不确定是否有效

关于对象(Element)的创建:

创建完project后创建对象,例如地板(plane),方块(cube),球(sphere),均在导航栏的GameObject -> 3DObject 中 利用xy轴对Cube,Sphere进行精准定位

关于设定玩家(player):

在选中需要设定为player的Element上inspector中的tag设定为player即可

关于物理模拟:

选中需要物理的Element然后依次 Component -> Physics -> Rigidbody 此时,若物理模拟的Element在空中的话,便会掉落到Plane上后停止运动

关于c#的脚本添加:

添加方式可以从项目上边的Create中创建,或者从导航栏中选 Assets -> Create -> c# script 来创建文件. 若有vs的话,可以直接双击利用vs来编辑,觉得vs太笨重的话,可以尝试用vscode,创建的文件位于project/Assets中. 希望script应用到的Element可以直接用鼠标将scirpt file 拖向项目表中的 Element上

关于材质(Material)的添加:

Create -> Material 然后可以点击inspector的 Main Maps -> Albedo 右边的输入框来选择需要的基本色 应用到哪个Element上只需要拖过去即可

关于游戏屏幕大小设定:

在Game -> Free Aspect 中选择或者点”+”自定义.(需要注意的是从最初开始都是在Sence上,Game在Sence的旁边,Sence在Create右边
