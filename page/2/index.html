SQLite3

sqlite3是一个很方便的数据库管理工具,但是常规使用依然建议使用MYSQL

sqlite3 下载界面
1.用sqlite3打开数据库,若不存在则创建数据库
sqlite3 DatabaseName.db .databases
显示打开的数据库
.exit 退出
.help 帮助

2.创建和删除表
sqlite3 create table <tablename> (Var1 Type1 , Var2 Type2...);
eg: create table user (ID INTEGER , name Text , height REAL);
创建一个名叫user的表,ID是数字列,name是文本列,height是浮点数列
.schema 显示表的结构
drop table <tablename>
eg:drop table user; 删除user表

3.插入,删除
insert into <tablename> values(Var1 Type1 , Var2 Type2...);
插入一条数据
delete from <tablename> where Var1=Value1 and/or Var2=Value2....;
删除一条数据Var1=Value1 并且/或者 Var2=Value2

4.查询,更新
select Var1,Var2... from <tablename>; select * from <tablename>;
从表中查询 Var1和Var2/*全部
Update <tablename> set Var1=Value1,Var2=Value2...;
更新,where语句可以用于删改查用于定位其中某一条或者某一些

14. 二分查找

描述

给定一个排序的整数数组（升序）和一个要查找的整数target,用O(logn)的时间查找到target第一次出现的下标（从0开始）,如果target不存在于数组中,返回-1.

样例

在数组 [1, 2, 3, 3, 4, 5, 10] 中二分查找3,返回2.

挑战

如果数组中的整数个数超过了2^32,你的算法是否会出错?

如果对二分查找比较陌生话,那么讲个小游戏,猜数字,在0-10000中随机取个数,让你猜,猜错了告诉你大了还是小了.

那么第一想法是什么?报中位数?对,不停的报中位数直到剩下最后一个数,这个数就是所需数,这个方法是比一个一个遍历要快非常多所需步骤也仅为log2n 回到正题

class Solution:
def binarySearch(self, nums, target):

    # write your code here
    start=0
    end=len(nums)
    mid=int(end/2)
    while start<=end:
        n_mid = nums\[mid\]
        if n_mid <target: #right
            start=mid+1
            mid=int((end+start)/2)
        elif n_mid >target: #left
            end=mid-1
            mid=int((end-start)/2)
        else:
            while nums\[mid\]==target:
                mid-=1
            return mid+1
    return -1

参考于这篇做了一些修正,在没有重复的情况下,else的部分去除.直接return mid+1

4. 丑数 II

描述

设计一个算法，找出只含素因子2，3，5 的第 _n_ 小的数。 符合条件的数如：1, 2, 3, 4, 5, 6, 8, 9, 10, 12...

我们可以认为1也是一个丑数

样例

如果n = 9， 返回 10

挑战

要求时间复杂度为O(_n_log_n_)或者O(_n_)

找了很久都没明白丑数几个意思,直到看到个公式: 2^i,3^j,5^k 10以内丑数为 n[2^i,3^j,5^k] 1 [0, 0, 0] 2 [1, 0, 0] 3 [0, 1, 0] 4 [2, 0, 0] 5 [0, 0, 1] 6 [1, 1, 0] 7 不存在 8 [3, 0, 0] 9 [0, 2, 0] 10 [1, 0, 1] 要返回的数是第n个,即输入9返回的并不是9(9是丑数但不是第九个),第9个是10 思路就是生成丑数数组,然后返回坐标为 n-1 的元素即可 Python3

class Solution:
“””
@param n: An integer
@return: the nth prime number as description.
“””
def nthUglyNumber(self, n):
L=[1]
i2,i3,i5=0,0,0
for j in range(n):
n2,n3,n5=L[i2]*2,L[i3]*3,L[i5]*5
m=min(n2,n3,n5)
L+=[m]
i2+=(m == n2)
i3+=(m == n3)
i5+=(m == n5)
print(L)
return L[n-1]

其中 i2+=(x) 当(x)为True时i2+=1 当(x)为False时i2+=0

Unity : Make a simple game

读完这篇教程而记 https://www.cnblogs.com/OctoptusLian/p/8418534.html

前言 unity版本2018.1.3.f1 安装unity的时候出现了个坑,以前安装过unity,license 认证死活过不去,然后重装解决了这个问题 upgrade license上依然出错,解决方法: 1 log out后再login 2 删掉C:/ProgramData中的Unity文件夹(不确定是否有效

关于对象(Element)的创建:

创建完project后创建对象,例如地板(plane),方块(cube),球(sphere),均在导航栏的GameObject -> 3DObject 中 利用xy轴对Cube,Sphere进行精准定位

关于设定玩家(player):

在选中需要设定为player的Element上inspector中的tag设定为player即可

关于物理模拟:

选中需要物理的Element然后依次 Component -> Physics -> Rigidbody 此时,若物理模拟的Element在空中的话,便会掉落到Plane上后停止运动

关于c#的脚本添加:

添加方式可以从项目上边的Create中创建,或者从导航栏中选 Assets -> Create -> c# script 来创建文件. 若有vs的话,可以直接双击利用vs来编辑,觉得vs太笨重的话,可以尝试用vscode,创建的文件位于project/Assets中. 希望script应用到的Element可以直接用鼠标将scirpt file 拖向项目表中的 Element上

关于材质(Material)的添加:

Create -> Material 然后可以点击inspector的 Main Maps -> Albedo 右边的输入框来选择需要的基本色 应用到哪个Element上只需要拖过去即可

关于游戏屏幕大小设定:

在Game -> Free Aspect 中选择或者点”+”自定义.(需要注意的是从最初开始都是在Sence上,Game在Sence的旁边,Sence在Create右边
